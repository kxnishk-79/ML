# 🚀 Machine Learning Projects Collection

A collection of beginner-to-intermediate level Machine Learning projects focused on real-world applications in the food and restaurant domain.

This repository demonstrates practical implementation of data preprocessing, model building, evaluation, and result interpretation.

---

## 📌 Projects Included

### ⭐ Task 1: Restaurant Rating Prediction

* Predicts restaurant ratings based on available features
* Includes data preprocessing, feature engineering, and regression modeling
* Focus on understanding model performance and avoiding data leakage

---

### 🍽️ Task 2: Restaurant Recommendation System

* Recommends restaurants based on user preferences
* Uses similarity-based approach for generating recommendations
* Clean and simple implementation with interpretable results

---

### 🌍 Task 3: Cuisine Classification

* Classifies restaurants into different cuisine categories
* Handles class imbalance and improves dataset quality
* Implements classification models with performance evaluation

---

## 🛠️ Tech Stack
