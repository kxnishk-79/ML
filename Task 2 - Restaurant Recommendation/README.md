A simple **content-based restaurant recommendation system** built using **Python and Pandas**.  
The system recommends restaurants based on **user preferences such as cuisine, city, budget, and rating**.

---

# 📌 Project Objective

Create a **restaurant recommendation system based on user preferences**.

The system filters restaurants from a dataset and recommends the best matches based on the user's requirements.

---

# ✨ Features

✔ Load and process a restaurant dataset  
✔ Handle missing values in the dataset  
✔ Accept user preferences through terminal input  
✔ Filter restaurants based on multiple criteria  
✔ Rank restaurants based on rating  
✔ Display clean and readable recommendations  

---

# 📊 Dataset Information

The dataset contains restaurant information such as:

| Column | Description |
|------|-------------|
| Restaurant Name | Name of the restaurant |
| City | Location of the restaurant |
| Cuisines | Types of cuisines served |
| Average Cost for Two | Estimated cost for two people |
| Aggregate Rating | Restaurant rating |
| Votes | Number of votes received |
